# Setup

Development environment setup scripts and configurations for Windows, macOS, and Ubuntu.

## Directories

- **[Windows](Windows/README.md)**: Setup scripts using Winget, Chocolatey, and Powershell.
- **[macOS](macOS/README.md)**: Setup scripts using Homebrew.
- **[Ubuntu](Ubuntu/README.md)**: Setup guides for Ubuntu.
- **[python](python/README.md)**: Python environment setup using Poetry and Pyenv.
- **[ai-agents](ai-agents/README.md)**: AI agent setup (Claude Code, Gemini CLI, Codex, Copilot CLI, Superpowers, BMAD, bkit, oh-my-claudecode, oh-my-codex, claude-hud, GSD).
